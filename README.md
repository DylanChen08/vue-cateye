#### 技术栈
前端
- Vue
- Vuex
- VueRouter
- Vue-cli
- Lodash
- axios
- BetterScroll
- vue-swiper-component

后端
- nodeJs
- Express
- Charles 收集数据
- MongoDB
- Mongoose

没有部署云服务器 , 因为很多还没完工。

本地运行方式:

前端：

```
git clone git@github.com:EaseCy/vue-cateye.git
cd vue-cateye
npm run serve
```
后端：
```
git clone git@github.com:EaseCy/vue-catEye-backend.git
cd vue-catEye-backend
npm run dev
```
